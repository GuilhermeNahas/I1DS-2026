# Explicação do projeto Urban Wear

## Estrutura criada

- `Urban Wear/urbanWear.html`
- `Urban Wear/style.css`
- `Urban Wear/explicacao.md`

Os arquivos `urbanWear.html` e `style.css` foram movidos para a pasta `Urban Wear` para manter o projeto organizado.

## O que foi feito no HTML

1. Usei uma estrutura semântica com `<header>`, `<main>`, `<section>` e `<footer>`.
2. Adicionei um cabeçalho fixo com o nome da loja e links de navegação internos.
3. Criei uma seção `hero` contendo:
   - imagem à esquerda;
   - formulário de login à direita;
   - botões de login social e link para cadastro.
4. Incluí uma seção de coleções com dois cards (feminino e masculino).
5. Adicionei uma seção de catálogo com quatro produtos em destaque.
6. Criei uma seção de cadastro com formulário simples.

## O que foi feito no CSS

1. Defini `body` com fundo preto e texto claro para o visual escuro do protótipo.
2. Usei `display: grid` para organizar o layout principal e os cards.
3. Adicionei bordas arredondadas e bordas translúcidas para o estilo moderno.
4. Configurei os inputs e botões para ficarem com aparência uniforme:
   - botões principais com fundo branco;
   - botões secundários com fundo translúcido.
5. Tornei o site responsivo com media queries para telas menores.

## Como abrir e testar

1. Abra o arquivo `Urban Wear/urbanWear.html` no navegador.
2. O CSS será carregado automaticamente porque o link está relativo (`href="style.css"`).
3. Você pode testar em diferentes larguras de tela para ver o layout responsivo.

## O que aprender com este projeto

- Use sempre caminho relativo para o CSS quando o HTML e o CSS estiverem na mesma pasta.
- Separe o conteúdo (HTML) do estilo (CSS).
- Prefira classes e estruturas semânticas para facilitar a leitura e manutenção.
- Para layouts modernos, `display: grid` é muito útil para organizar colunas.
- Use `@media` para adaptar o layout em dispositivos menores.

## Observações extras

- A imagem do protótipo foi simulada usando imagens externas. Para deixar o projeto completo, você pode trocar essas imagens por outras do seu Figma ou imagens locais.
- Se quiser, posso também adicionar uma segunda página de cadastro ou uma navegação mais completa."
